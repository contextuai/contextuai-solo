# Video 1: First Launch & Setup

**Duration:** 3 minutes
**Goal:** Get the user from zero to their first AI conversation in under 3 minutes.

---

## Opening (0:00 - 0:15)

**On screen:** Solo logo animation, then tagline: *"The AI team behind every solo business"*

**Voiceover:**
> "ContextuAI Solo gives you an entire team of AI business experts — running privately on your desktop. Let's get you set up in under 3 minutes."

---

## Scene 1: Download & Install (0:15 - 0:45)

**On screen:** Browser → GitHub Releases page

**Steps:**
1. Show the GitHub releases page with download buttons
2. Click the Windows `.exe` installer (mention macOS `.dmg` and Linux `.deb` alternatives)
3. Run the installer — show the SmartScreen bypass ("More info" → "Run anyway")
4. App icon appears on desktop

**Voiceover:**
> "Head to the releases page, download the installer for your platform, and run it. Windows might show a SmartScreen warning — that's normal for new open-source software. Click 'More info', then 'Run anyway'. You'll only see this once."

---

## Scene 2: First Launch (0:45 - 1:15)

**On screen:** App opens → Setup Wizard

**Steps:**
1. App launches with the setup wizard
2. Show two paths: "Download a free local model" vs "Enter an API key"
3. Choose "Download local model" path for the demo

**Voiceover:**
> "On first launch, Solo walks you through setup. You have two choices — download a free AI model that runs entirely on your machine, or enter an API key for a cloud provider like Anthropic or OpenAI. Let's go local — it's free and completely private."

---

## Scene 3: Download Your First Model (1:15 - 1:50)

**On screen:** Model Hub → Download Qwen 3 8B

**Steps:**
1. Show the Model Hub with recommended models
2. Click **Download** on Qwen 3 8B (or Gemma 3 1B for a faster demo)
3. Show "Preparing download..." → progress bar with percentage and MB counters
4. Model shows "Installed" badge when complete

**Voiceover:**
> "Solo recommends models based on your hardware. I'll download Qwen 3 8B — it's great for business writing and analysis. You can see the progress percentage and file size as it downloads. Once it's done, you'll see the 'Installed' badge."

---

## Scene 4: Your First Chat (1:50 - 2:40)

**On screen:** Navigate to Chat → Send a message

**Steps:**
1. Click **Chat** in the sidebar
2. Model is auto-selected from the one just downloaded
3. Type: "Write me a 3-point elevator pitch for a freelance marketing consultancy"
4. Show streaming response appearing word by word
5. Point out the markdown formatting, bold text, bullet points

**Voiceover:**
> "Head to Chat — your model is already selected. Let's try something real. I'll ask it to write an elevator pitch for a freelance marketing consultancy. Watch how the response streams in real-time with formatted text. That's running entirely on your machine — no internet needed."

---

## Scene 5: What's Next (2:40 - 3:00)

**On screen:** Quick montage of Agent Library, Crews, Workspace

**Voiceover:**
> "You're up and running. But chatting is just the beginning. Solo has 81 business agents, multi-agent crews, and a workspace for complex projects. We'll cover all of that in the next videos. For now — explore, ask questions, and make Solo your own."

**End card:** "Next: Model Hub Deep Dive" + Subscribe/Follow CTA

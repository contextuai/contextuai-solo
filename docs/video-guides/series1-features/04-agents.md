# Video 4: Meet Your Agent Library

**Duration:** 3 minutes
**Goal:** Show users the 81-agent library and how to find, understand, and customize agents.

---

## Opening (0:00 - 0:15)

**On screen:** Agent Library page with cards visible

**Voiceover:**
> "Solo comes with 81 pre-built business agents — specialists across marketing, finance, legal, HR, design, and more. Think of them as expert colleagues you can assign to any task. Let's meet the team."

---

## Scene 1: Browsing the Library (0:15 - 0:55)

**On screen:** Agent Library grid

**Steps:**
1. Show the full grid — scroll through to show the volume (81 agents)
2. Point out a card: agent name, role badge (Analyst, Writer, Researcher), description, tools
3. Click a role filter pill — e.g., "Analyst" — show the count badge (e.g., "Analyst (15)")
4. Click "All" to reset
5. Type "pricing" in search — show Pricing Strategist appearing
6. Clear search

**Voiceover:**
> "Each agent has a name, role, description, and tools it can use. Filter by role — there are 15 analysts alone. Or search by keyword. Type 'pricing' and the Pricing Strategist pops right up. These filters combine, so you can search within a role."

---

## Scene 2: Agent Details (0:55 - 1:30)

**On screen:** Click an agent card → detail panel slides in

**Steps:**
1. Click "Content Strategist" card
2. Show the detail panel: name, role, description
3. Scroll to the **system prompt** — highlight that this is what makes the agent an expert
4. Show the tools section (Web Search, Database, etc.)
5. Show the model override option

**Voiceover:**
> "Click any agent to see its full profile. The key is the system prompt — this is a detailed instruction set that makes the AI behave like a content strategist. It defines expertise, methodology, and how to structure deliverables. You can also see which tools the agent has access to, and override the model if you want."

---

## Scene 3: Customizing an Agent (1:30 - 2:05)

**On screen:** Edit fields in the detail panel

**Steps:**
1. Edit the system prompt — add a line: "Always include ROI estimates in your recommendations"
2. Toggle a tool on/off
3. Click **Save**
4. Show the card updated

**Voiceover:**
> "Every agent is customizable. Add a line to the system prompt to tailor it to your business — like 'Always include ROI estimates.' Toggle tools on or off. Hit save. The agent is now yours — same expertise, personalized for how you work."

---

## Scene 4: Creating a Custom Agent (2:05 - 2:40)

**On screen:** Click "Create Agent" → fill the form

**Steps:**
1. Click **Create Agent**
2. Enter name: "Proposal Writer"
3. Select role: "Writer"
4. Show the suggested template appearing — click "Use suggested template"
5. Add a custom line to the prompt
6. Check tools: Web Search
7. Click **Create Agent**
8. Show the new agent in the grid

**Voiceover:**
> "Need something custom? Click Create Agent, give it a name and role. Solo suggests a template based on the role — use it as a starting point. Customize the prompt, pick your tools, and create. Your new agent joins the library alongside the 81 built-in ones."

---

## Scene 5: Where Agents Work (2:40 - 3:00)

**On screen:** Quick flash of Crews wizard (agent selection) and Workspace wizard (agent selection)

**Voiceover:**
> "Agents aren't just for browsing. You'll use them in Crews to build multi-agent teams, and in Workspace to run projects. The library is your roster — the real action happens when you put them to work. That's what we'll cover next."

**End card:** "Next: Personas — Your AI Personalities"

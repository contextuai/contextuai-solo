# Video 3: Chat Like a Pro

**Duration:** 3 minutes
**Goal:** Show all chat features — sessions, models, personas, formatting, shortcuts.

---

## Opening (0:00 - 0:10)

**Voiceover:**
> "Chat is where you'll spend most of your time in Solo. Let's go beyond basic messaging and use every feature it has."

---

## Scene 1: Starting a Conversation (0:10 - 0:40)

**On screen:** Chat page, empty state

**Steps:**
1. Select a model from the dropdown
2. Type a message: "Help me write a proposal introduction for a data analytics consulting engagement"
3. Show streaming response with formatted output
4. Point out the session title auto-generated in the sidebar

**Voiceover:**
> "Pick your model, type your message, and hit Enter. The response streams in real-time. Notice the sidebar — Solo auto-names your session from the first message so you can find it later."

---

## Scene 2: Managing Sessions (0:40 - 1:15)

**On screen:** Sidebar interactions

**Steps:**
1. Press **Ctrl+N** to create a new session
2. Start a different conversation
3. Click back to the first session — show messages preserved
4. Click the session title in the header — rename it
5. Hover over a session → show the three-dot menu → Archive
6. Use the search box to filter sessions

**Voiceover:**
> "Ctrl+N starts a new session. Click any session to switch back — your history is preserved. Rename sessions by clicking the title. Archive old ones to keep things tidy, or search to find past conversations."

---

## Scene 3: Using Personas (1:15 - 1:45)

**On screen:** Persona dropdown in chat input

**Steps:**
1. Click the persona dropdown
2. Select a persona (e.g., "Financial Analyst")
3. Ask: "What are the key metrics I should track for a SaaS business?"
4. Show how the response is more structured and domain-specific
5. Switch to "None (default)" and ask the same question — show the difference

**Voiceover:**
> "Personas change how the AI thinks. Select 'Financial Analyst' and ask about SaaS metrics — you get structured, domain-specific advice. Without a persona, you get a general answer. It's like talking to a specialist vs. a generalist."

---

## Scene 4: Rich Formatting (1:45 - 2:15)

**On screen:** Responses with formatting

**Steps:**
1. Ask: "Write a Python function to calculate compound interest, then explain the formula"
2. Show the code block with syntax highlighting and the **copy button**
3. Show bold text, bullet points, headers in the explanation
4. Click the "Thinking" collapse section (if available with the model)

**Voiceover:**
> "Responses come with full markdown — code blocks with syntax highlighting and a copy button, headers, bullet points, bold text. Some models even show their reasoning in a collapsible 'Thinking' section."

---

## Scene 5: Power Tips (2:15 - 3:00)

**On screen:** Quick demo of each tip

**Steps:**
1. **Shift+Enter** for multi-line input — compose a longer prompt
2. **Stop button** — click to cancel a long response mid-stream
3. **Dark mode toggle** — click sun/moon in header
4. **Model switching** — change model mid-conversation for a different perspective

**Voiceover:**
> "A few power tips: Shift+Enter lets you write multi-line prompts. The stop button cancels generation if you've seen enough. Toggle dark mode from the header. And you can switch models mid-conversation — your history stays, you just get a fresh perspective."

**End card:** "Next: Meet Your Agent Library"

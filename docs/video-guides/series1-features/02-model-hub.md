# Video 2: Model Hub Deep Dive

**Duration:** 3 minutes
**Goal:** Help users understand local vs cloud models and pick the right one for their needs.

---

## Opening (0:00 - 0:10)

**On screen:** Model Hub page with models visible

**Voiceover:**
> "Solo's Model Hub is where you manage your AI models — free local ones or cloud providers with your own API keys. Let's find the right model for you."

---

## Scene 1: Local vs Cloud — What's the Difference? (0:10 - 0:45)

**On screen:** Side-by-side comparison graphic

**Voiceover:**
> "Local models run on your CPU — they're free, completely private, and work offline. Cloud models are faster and more capable, but they need an internet connection and an API key you pay for. The good news? You can use both. Switch anytime."

**On screen:** Show the AI Mode toggle in the sidebar (Local green badge / Cloud blue badge)

> "This toggle switches between local and cloud mode. Or click 'All models' in Chat to see everything."

---

## Scene 2: Browsing the Model Catalog (0:45 - 1:20)

**On screen:** Model Hub → Discover tab

**Steps:**
1. Show the catalog grid with model cards
2. Point out key info on each card: model name, parameter size, RAM required, categories (chat, coding, reasoning)
3. Show the RAM indicator — green (fits), yellow (tight), red (too large)
4. Click a category filter (e.g., "Coding")
5. Show the "Recommended for you" section at the top

**Voiceover:**
> "The Discover tab shows every model Solo supports — organized by category. Each card tells you the model name, size, and how much RAM it needs. Green means it'll run comfortably on your machine. Solo even recommends models based on your hardware at the top."

---

## Scene 3: Downloading a Local Model (1:20 - 1:50)

**On screen:** Click Download on a model

**Steps:**
1. Click **Download** on a model
2. Show "Preparing download..." phase
3. Show progress: percentage + MB counters (e.g., "42% — 680 / 1620 MB")
4. Show "Installed" badge when complete
5. Switch to the **Installed** tab to see it listed

**Voiceover:**
> "Click Download and Solo fetches the model from HuggingFace. You'll see the percentage and exact megabytes as it progresses. Once it's installed, it appears in your Chat model dropdown automatically."

---

## Scene 4: Adding a Cloud Provider (1:50 - 2:25)

**On screen:** Settings → AI Providers

**Steps:**
1. Navigate to Settings → AI Providers
2. Expand the Anthropic card
3. Paste an API key
4. Click **Test Connection** — show green checkmark
5. Go back to Chat — show Claude models in the dropdown

**Voiceover:**
> "For cloud models, go to Settings, expand a provider card, and paste your API key. Click Test Connection to verify. Now Claude, GPT-4, or Gemini models appear in your Chat dropdown. You pay the provider directly — Solo never touches your key beyond routing requests."

---

## Scene 5: Picking the Right Model (2:25 - 3:00)

**On screen:** Table/graphic showing use case recommendations

**Voiceover:**
> "Quick guide: For everyday business writing, Qwen 3 8B or Claude Sonnet are great choices. For coding, try Qwen 2.5 Coder or DeepSeek R1. For complex reasoning, use a larger model like Qwen 3 14B or GPT-4o. And if you just need quick answers, the smaller 1B models are instant."

**On screen:** Show the "View on HuggingFace" link on a model card

> "Every model card links to its HuggingFace page if you want technical details. Next up — let's put these models to work in Chat."

**End card:** "Next: Chat Like a Pro"

# Video 3: Chat Like a Pro

> **Director's Context:** ContextuAI Solo is a desktop AI app with local models. This video covers the Chat interface — the primary way users interact with AI. The audience is business professionals who are familiar with ChatGPT but new to Solo. The goal is to show them that Solo's chat is just as polished, plus it has unique features like multiple sessions, model switching, and persona integration.

**Duration:** 3 minutes
**Goal:** Show all chat features — sessions, model switching, personas, formatting, and keyboard shortcuts.

---

## Opening (0:00 - 0:10)

**On screen:** Chat interface with a conversation in progress
![Screenshot: The Chat interface showing a conversation with formatted AI response including bold text, bullet points, and structured output.](scene1-chat-interface.png)

**Voiceover:**
> "Chat is where you'll spend most of your time in Solo. Let's make sure you know every trick."

---

## Scene 1: Starting a Conversation (0:10 - 0:40)

**On screen:** Click Chat in sidebar → empty chat → model dropdown → type message

**Voiceover:**
> "Click Chat in the sidebar. You'll see a model dropdown at the top — it defaults to your best installed model. Type your message and hit Enter. The response streams in real-time with full markdown formatting — bold text, bullet points, code blocks, tables. Everything renders beautifully."

**Key point for NotebookLM:** Emphasize the streaming response — it feels responsive even on CPU because tokens appear one at a time. This is a desktop app, not a web wrapper.

---

## Scene 2: Managing Sessions (0:40 - 1:15)

**On screen:** Show multiple sessions in the sidebar, Ctrl+N shortcut, rename, archive

**Voiceover:**
> "Every conversation is saved as a session. Hit Ctrl+N to start a new one without losing your current thread. You can rename sessions to keep them organized — 'Q2 Marketing Plan', 'Client Proposal Draft'. Old sessions can be archived to keep your sidebar clean. Nothing is ever deleted unless you want it to be."

---

## Scene 3: Switching Models Mid-Chat (1:15 - 1:40)

**On screen:** Model dropdown → switch from local model to cloud model mid-conversation

**Voiceover:**
> "Here's something powerful — you can switch models in the middle of a conversation. Start with a local model for brainstorming, then switch to Anthropic Claude for the final polished version. The entire conversation history carries over. Mix local and cloud models freely."

---

## Scene 4: Using Personas in Chat (1:40 - 2:10)

**On screen:** Persona dropdown → select a persona → chat behavior changes

**Voiceover:**
> "Personas change how the AI behaves. Select a persona from the dropdown and the AI takes on that personality — a marketing expert, a financial analyst, a database connector. We'll cover personas in depth in the next video, but know that they're always one click away in Chat."

---

## Scene 5: Power Tips (2:10 - 2:45)

**On screen:** Show each shortcut being used

**Voiceover:**
> "A few power tips. Shift+Enter gives you multi-line input — great for pasting lists or writing complex prompts. The AI formats responses in rich markdown, so ask for tables, numbered lists, or code and it delivers. You can also stop a response mid-stream if it's going in the wrong direction — click the stop button or hit Escape."

**Key point for NotebookLM:** The stop/cancel feature is important — it saves time with local models which can be slower. Users aren't stuck waiting for a bad response to finish.

**Open-sourcing ContextuAI Solo: what I learned shipping a complex desktop app to the community.**

Today I'm releasing the full source code for ContextuAI Solo — a desktop AI assistant with 93+ business agents, 35+ local models, and multi-agent crew orchestration.

**The stack (all open source):**
- Tauri v2 (Rust) — MIT licensed
- React 19 — MIT licensed
- FastAPI — MIT licensed
- SQLite — public domain
- llama-cpp-python — MIT licensed
- Every GGUF model — open weight

**What I'm looking for from the community:**

- **Agent contributors:** The 93 agents are markdown files with system prompts. Easy to add new ones or improve existing prompts. No coding required.
- **Model suggestions:** Which GGUF models should be in the Model Hub? What sizes work best for business tasks?
- **Platform testers:** I've tested thoroughly on Windows. macOS and Linux could use more eyes.
- **Localization:** The agents are English-only. Multilingual agents would be a huge unlock.
- **Blueprint authors:** Workflow templates are markdown files. Share your business workflows.

**Good first issues** are labeled and ready. The codebase is documented with inline comments and a comprehensive CLAUDE.md.

GitHub: https://github.com/contextuai/contextuai-solo
